Resume

Dalam mata kuliah algoritma struktur data saya mengenal bagaimana caranya mengatur data yang disebut struktur data tapi sebelum itu saya belajar bagaimana cara membuat algoritma yaitu membuat urutan langkah yang logis untuk memecahkan suatu masalah contohnya seperti kita makan pun ada urutan nya : mulai--> mengambil piring dan sendok--> mengambil nasi-->mengambil lauk-->end. Lalu ada struktur data yaitu cara kita menyimpan, mengatur dan mengelola data tersebut dan ada dua struktur data yang pertama ada sturktur data sederhana dan yang kedua ada struktur data majemuk. Dalam struktur data pertama yaitu struktur data sederhana ada yang namanya array,tapi sebelum belajar array saya belajar tentang tipe data yang sangat penting ketika nanti menggunakan array Tipe data sendiri ini menandakan jenis data yang akan disimpan dengan menyesuaikan kegunaan memori penyimpanan, mengapa tipe data sangat di butuhkan? Karena data banyak macam isi nya seperti angka, huruf, karakter dll. Maka dari itu sangat dibutuhkan tipe data ini. Tipe data yang digunakan dalam c++ yang sedang saya pelajari sekarang ada integer(bilangan bulat), short int(bilangan bulat yang jangkauan nya lebih pendek), long int(bilangan bulat yang jangkauan nya lebih panjang dari integer), boolean(nilai kebenaran yaitu false or true), float(bilangan pointing atau koma),

double(sama seperti float namun memiliki jangkauan dua kali dari float) , long double(sama dengan double hanya jangkauan nya lebih lebar), char(berisi karakter atau huruf saja), wchar_t(sama dengan char namun jangkauan nya lebih lebar). Setelah kita tau apa itu tipe data, maka kita akan gunakan tipe data tersebut pada array. Array(larik) merupakan proses penyimpanan data dengan jumlah yang banyak secara efisien dan mudah, Pada umumnya tipe data yang di gunakan oleh array ini adalah int, char, string, float, dan double. Array terbagi menjadi dua macam yaitu array satu dimensi dan array multi dimensi. Array satu dimensi :proses penyimpanan data secara berurutan, kalau dalam bahasa pemrograman seperti: tipe-data nama-array [ukuran]; misalnya saya ingin menyimpan data absen sebanyak 5 dengan tipe data yang sama saya tidak perlu menuliskan nya satu-satu, cukup dengan menggunakan array satu dimensi maka penulisan nya int absen [5]; Yang kedua yaitu array multidimensi Array multidimensi adalah alat yang kuat dalam pemrograman untuk menyimpan dan mengelola data dalam bentuk tabel dengan lebih dari satu dimensi. Memahami cara kerjanya memungkinkan Anda untuk secara efektif memanipulasi data dan melakukan operasi yang diperlukan dalam aplikasi Anda.
